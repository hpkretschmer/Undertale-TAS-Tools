# Undertale TAS Tools

The TASing tools that I often use.

## Current tools

- `rngtools`: general-purpose rng manipulation tools
- `maddummytools`: 4-cycle and tear offset finder
- `aronwoshua`: Aaron/Woshua step count manip finder, by Svool
- `ltm_merge_tool`: LTM stitcher and seed resync, by OceanBagel
- `musicmarkers`: GD-style music sync marker insertion
- `new_blue`: Papyrus blue attack offset finder, by Svool
- `torielsolver`: Toriel manip finder, by Svool
- `utusiner`: UTU transformation cutscene siner offset finder
